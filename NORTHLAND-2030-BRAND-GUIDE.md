# Northland Computers 2030 — Brand Guide

## Positioning
Northland Computers is a technology consulting and managed services firm that helps organizations make confident technology decisions and execute difficult work.

## Primary message
**Technology decisions made with confidence. Projects delivered with accountability.**

## Brand pillars
1. Execution
2. Response
3. Honest recommendations
4. Documentation
5. Client ownership
6. Long-term planning

## Voice
Professional, calm, direct, experienced, practical, and never sales-heavy.

## Visual system
- Navy: `#06172F`
- Electric blue: existing Tailwind `electric`
- Cyan: existing Tailwind `cyan`
- Signal green: existing Tailwind `signal`
- White and slate neutrals

## Calls to action
- Schedule a Consultation
- See Proven Results
- Explore Solution
- Discuss Your Environment

## Content rule
Prove capability with measurable outcomes. Avoid generic claims such as “best-in-class,” “cutting-edge,” and “world-class.”
