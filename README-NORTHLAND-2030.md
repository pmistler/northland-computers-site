# Northland Computers 2030

Production-ready Next.js website for Northland Computers.

## Included in this release
- Updated flagship homepage with business-outcome positioning
- Managed IT, cybersecurity, infrastructure, consulting, AI, cloud, network, server, website, and emergency support pages
- Industry landing pages
- Project portfolio with measurable results
- Why Northland, About, Service Plans, Contact, Resources, and Technology Leadership pages
- Client-ownership and documentation messaging
- SEO metadata, schema, sitemap, and redirects
- Bachelor of Science in Computer Science correction

## Deploy
1. Upload the project contents to the GitHub repository root.
2. Keep `package-lock.json` committed.
3. Vercel should detect Next.js automatically.
4. Install command: `npm install --no-audit --no-fund`
5. Build command: `npm run build`
6. Node version: 20.x for this tested release.

The project was successfully production-built before packaging.
